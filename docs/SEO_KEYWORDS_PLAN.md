# Plano de palavras‑chave (SEO + Ads)

## Princípios
- 1 intenção principal por página (evita canibalização).
- Usar variações em H2/FAQ e links internos (sem repetição artificial).
- Reforçar cluster: **Comprar / Preço / Criador confiável** apontam para **/filhotes** e **/reserve-seu-filhote**.

## Páginas pilares (alta intenção)
### /comprar-spitz-anao
- Principal: comprar spitz anão
- Variações: comprar spitz alemão anão; comprar lulu da pomerânia; spitz anão com pedigree; entrega de filhote
- Papel: explicar processo + reduzir fricção + levar para catálogo e WhatsApp.

### /preco-spitz-anao
- Principal: preço spitz anão
- Variações: quanto custa spitz anão; valor spitz anão; fatores que influenciam preço
- Papel: educar investimento + qualificar lead + direcionar para catálogo e WhatsApp.

### /criador-spitz-confiavel
- Principal: criador spitz confiável
- Variações: canil spitz anão confiável (termo do usuário); documentação; sinais de alerta
- Papel: construir confiança + checklist objetivo + CTA.

## Catálogo (intenção transacional)
### /filhotes
- Principal: filhotes spitz anão
- Variações: filhote spitz anão com pedigree; filhotes disponíveis; entrega

## Blog (volume + interlinking)
- Posts sobre dúvidas reais (ex.: late muito, solta pelo, apartamento, crianças, banho/tosa) devem linkar para as 3 páginas pilares.

## Âncoras internas (UX + SEO)
- Cada página pilar expõe um mini sumário no topo com links para seções (faixas, checklist, FAQ, formulário/CTA).
